Help & Info

Everything you need to know about SplitEase

🚀 Quick Start — 4 Steps

1
Add Members

Go to the 👥 Members tab → click + Add Member. Add every person who shares expenses (roommates, family, friends). Each member gets a color avatar.

2
Create Groups (optional but recommended)

Go to 🏠 Groups tab → click + Create Group. Name it (e.g. "Home Bills", "Road Trip") and tick only the members who belong to that group. Like Splitwise groups.

3
Add Expenses

Go to 💸 Expenses tab → click + Add Expense. Fill in title, amount, who paid, date, category, and which members split it. Attach a bill photo if you like.

4
Check Balances & Settle Up

Go to ⚖️ Balances tab. See who owes how much. The app calculates the minimum number of payments needed to settle all debts (e.g. instead of 6 transfers it might suggest 2).

💾 Data Storage — How to Never Lose Your Data

🌐
Browser Storage (default)
Data saved inside your browser. Lost if you clear browser cache, use incognito, or switch browsers.
📁
Local File (recommended)
Click 💾 Browser Only in the header → link a .json file on your laptop. Every change auto-saves. Works across browser clears.
📤
Export / Import (manual backup)
Use 📤 Export to download a backup JSON file anytime. Use 📥 Import to restore it. Good for transferring between devices.
💡 Best practice: Link a local file AND do a monthly Export as an extra backup. Store the export in Google Drive or email it to yourself.

📖 Features Guide

💸
Expense Splits

Equal split — divides the amount equally among selected members (rounding handled automatically). Custom split — you enter each person's share manually; must add up to the total.

🏠
Groups

Create groups to organize expenses (Home, Vacation, etc.). Each group has its own members — only those members appear in the split when you select that group on an expense. Edit group members anytime via the ✏️ button on the group card.

📷
Bill Photos

Attach a receipt photo to any expense. The app compresses it automatically (a 4 MB phone photo becomes ~100 KB). Tap the thumbnail on any expense card to view it full size.

📊
Charts & Filters

Dashboard shows a category donut chart and spending bar chart. Use the Filter Charts panel to see spending for a specific group or for specific members. Toggle Daily / Monthly / Yearly for the bar chart.

💡
Spending Insights

The insights panel compares this month vs last month, projects your end-of-month total, shows your 3-month average, and flags your top spending category. Green = under budget, red = over.

⚖️
Debt Minimization

The Balances tab uses a greedy algorithm to find the fewest possible payments to settle all debts. Example: 4 people with 6 cross-debts might only need 3 bank transfers to fully settle up.

🏪
Store / Vendor Field

When adding an expense, the Store field has 60+ US retailers as autocomplete suggestions (Walmart, Costco, Target, McDonald's, Netflix, etc.). Just start typing.

Method A Save data file to Google Drive (for PC auto-sync)
1
Install Google Drive for Desktop — go to drive.google.com → click Download Drive for Desktop → install and sign in with your Google account
2
After install, a Google Drive folder appears in Windows Explorer (usually listed in the sidebar as "Google Drive" or shows as G:\My Drive\)
3
Move or copy your splitease-data.json file into that Google Drive folder — it will automatically sync to the cloud ☁️
4
In the app, click 💾 Browser Only in the header → Open existing data file → navigate to your Google Drive folder → select the .json file
5
Done! Every expense you add now auto-saves AND syncs to Google Drive. Open on any other Windows/Mac PC — same steps 3–4, pick the same file ✓
⚠️ On iPhone — Google Drive app cannot run the HTML app directly. Use Method B or C for iPhone access.
Method B iCloud Drive — best way to access on iPhone

Part 1 — Set up on your Windows PC

1
Install iCloud for Windows — open Microsoft Store → search "iCloud" → Install → sign in with your Apple ID
2
An iCloud Drive folder now appears in Windows Explorer. Copy your entire expense-splitter folder into it — it syncs to Apple's cloud
3
In the app, click 💾 → link your splitease-data.json from inside the iCloud Drive folder — changes will sync to your iPhone's Files app

Part 2 — Access on iPhone

4
On your iPhone, open the Files app → tap iCloud Drive → open the expense-splitter folder → tap index.html
5
A prompt appears — tap "Open in Safari" (or tap Share icon → Safari). The full SplitEase app opens and runs ✓
6
To load your data on iPhone: In the app, tap 📥 Import → Files → iCloud Drive → select your splitease-data.json
7
To send data back from iPhone to PC: tap 📤 Export on iPhone → save to iCloud Drive → on PC, tap 📥 Import in the app
✅ This works! The app runs fully in Safari from the Files app. All features work except the 💾 auto-save file button (use Import/Export instead on iPhone).
Method C ⭐ GitHub Pages — free URL, works on every device, no copying

Your app code already has 8 git commits saved locally. You just need to push it to GitHub and enable Pages — takes about 5 minutes.

Step 1 — Create your free GitHub account & repository

1
Go to github.com → click Sign up → create a free account (remember your username)
2
After login, click the + icon (top right) → New repository → Name: splitease → set to Public → click Create repository

Step 2 — Push your code (run these in Command Prompt inside the expense-splitter folder)

# Replace YOUR_USERNAME with your actual GitHub username
git remote add origin https://github.com/YOUR_USERNAME/splitease.git
git branch -M main
git push -u origin main
# It will ask for your GitHub username and password
# (use a Personal Access Token as password — see Step 3)

Step 3 — Create a GitHub Personal Access Token (used as your password)

1
On GitHub → click your profile photo (top right) → Settings → scroll down → Developer settings
2
Click Personal access tokens → Tokens (classic) → Generate new token (classic)
3
Give it a name (e.g. "splitease"), set expiration to No expiration, tick the repo checkbox → click Generate token
4
Copy the token immediately (you won't see it again). Paste it as your password when the git push command asks

Step 4 — Enable GitHub Pages (get your live URL)

1
Go to your repository on GitHub → click Settings tab → scroll down to Pages in the left sidebar
2
Under Source → select Deploy from a branch → Branch: main → folder: / (root) → click Save
3
Wait ~2 minutes → refresh the page → your URL appears: https://YOUR_USERNAME.github.io/splitease/
4
Open that URL on your iPhone in Safari — the full app loads! Bookmark it on your home screen for quick access 📌
✅ After this setup, your app is always at that URL. No copying folders. No USB cables. Open from any browser, any device, anywhere in the world.
⚠️ Data sync between devices: Each device has its own storage. Use 📤 Export on one device → 📥 Import on another to move data. The 💾 auto-save file button only works on Chrome/Edge on a PC.
