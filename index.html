Help & Info

Everything you need to know about SplitEase

🚀 Quick Start — 4 Steps

1
Add Members

Go to the 👥 Members tab → click + Add Member. Add every person who shares expenses (roommates, family, friends). Each member gets a color avatar.

2
Create Groups (optional but recommended)

Go to 🏠 Groups tab → click + Create Group. Name it (e.g. "Home Bills", "Road Trip") and tick only the members who belong to that group. Like Splitwise groups.

3
Add Expenses

Go to 💸 Expenses tab → click + Add Expense. Fill in title, amount, who paid, date, category, and which members split it. Attach a bill photo if you like.

4
Check Balances & Settle Up

Go to ⚖️ Balances tab. See who owes how much. The app calculates the minimum number of payments needed to settle all debts (e.g. instead of 6 transfers it might suggest 2).

💾 Data Storage — How to Never Lose Your Data

🌐
Browser Storage (default)
Data saved inside your browser. Lost if you clear browser cache, use incognito, or switch browsers.
📁
Local File (recommended)
Click 💾 Browser Only in the header → link a .json file on your laptop. Every change auto-saves. Works across browser clears.
📤
Export / Import (manual backup)
Use 📤 Export to download a backup JSON file anytime. Use 📥 Import to restore it. Good for transferring between devices.
💡 Best practice: Link a local file AND do a monthly Export as an extra backup. Store the export in Google Drive or email it to yourself.

📖 Features Guide

💸
Expense Splits

Equal split — divides the amount equally among selected members (rounding handled automatically). Custom split — you enter each person's share manually; must add up to the total.

🏠
Groups

Create groups to organize expenses (Home, Vacation, etc.). Each group has its own members — only those members appear in the split when you select that group on an expense. Edit group members anytime via the ✏️ button on the group card.

📷
Bill Photos

Attach a receipt photo to any expense. The app compresses it automatically (a 4 MB phone photo becomes ~100 KB). Tap the thumbnail on any expense card to view it full size.

📊
Charts & Filters

Dashboard shows a category donut chart and spending bar chart. Use the Filter Charts panel to see spending for a specific group or for specific members. Toggle Daily / Monthly / Yearly for the bar chart.

💡
Spending Insights

The insights panel compares this month vs last month, projects your end-of-month total, shows your 3-month average, and flags your top spending category. Green = under budget, red = over.

⚖️
Debt Minimization

The Balances tab uses a greedy algorithm to find the fewest possible payments to settle all debts. Example: 4 people with 6 cross-debts might only need 3 bank transfers to fully settle up.

🏪
Store / Vendor Field

When adding an expense, the Store field has 60+ US retailers as autocomplete suggestions (Walmart, Costco, Target, McDonald's, Netflix, etc.). Just start typing.

📱 Using on Other Devices

💻
Another Laptop / PC
  1. Copy the entire expense-splitter folder to the new laptop
  2. Also copy your splitease-data.json file
  3. Open index.html in Chrome or Edge
  4. Click 💾 in the header → Open existing data file → select your .json
  5. All your data loads instantly ✓
🤖
Android Phone
  1. Copy the expense-splitter folder to your phone (USB or cloud)
  2. Open Chrome → type file:/// in address bar → navigate to the folder → tap index.html
  3. Use 📤 Export on laptop → 📥 Import on phone to load your data
  4. Note: The 💾 local file feature may not be available on mobile Chrome
☁️
Best: Cloud File (any device)
  1. Save your splitease-data.json inside Google Drive, OneDrive, or Dropbox
  2. On any computer, open the app → click 💾 → link the file from your cloud folder
  3. Changes save to the cloud file → accessible from any laptop automatically
  4. Works like a shared data file across all your computers
⚠️ iPhone / iPad: Safari restricts local file access. Recommended workaround — use 📤 Export on your laptop, email the JSON to yourself, open on iPhone and use 📥 Import.

💡 Tips & Tricks

⌨️
Escape key closes any open modal instantly.
🔢
Click the Expenses or Groups stat card on the Dashboard to jump directly to that tab.
🏠
Selecting a group on the Add Expense form automatically pre-fills only that group's members in the Split section.
👁️
Click View Expenses → on any group card to jump to the Expenses tab filtered to that group only.
🔍
Use the Search + filters on the Expenses tab to find expenses by name, category, group, or date range.
✏️
You can edit group members anytime — tap ✏️ on a group card. Existing expenses are not affected.
📊
Use By Member(s) chart filter to see spending for just yourself — useful for tracking personal expenses in a shared household.
💾
Always set up the local file (💾 in header) on first use. It takes 10 seconds and protects your data permanently.
